Drawing with Data

Die nachfolgenden Aufgaben orientieren sich an dem Kapitel 6 des Buches Interactive Data Visualization for the Web von Scott Murray, erschienen im O'Reilly-Verlag. Eine Online-Version ist auf der Webressource von O'Reilly verfügbar (siehe http://chimera.labs.oreilly.com/books/1230000000345/ch06.html).

Die Elemente dieser Seite sind vorab mit einem Style versehen, sodass Sie den Anweisungen folgen sollten, um zum angestrebten Ergebnis zu kommen.

Zusammengefasste Informationen des Kapitels finden Sie unter dem Link https://github.com/mirar/Visualistik_DrawWithData/wiki

Aufgabe 1 - Balkendiagramm

  1. Erzeugen Sie mit Hilfe der Ihnen bekannten Funktionen
    • select()
    • append()
    • attr()
    einen senkrechten Balken, der folgender HTML-Syntax entspricht: 
    <div class="bar"></div>
  2. Übernehmen Sie die Änderung aus der vorhergehenden Aufgabe und binden Sie ein Datenarray ein, an welches Sie Balken, mit dem oben genannten Klassen-Attribut, binden.
  3. Übernehmen Sie die Änderung aus der vorhergehenden Aufgabe und passen Sie nun die Balkenhöhe an die Werte des Datenarrays an.
  4. Zusatz: Füllen Sie ein weiteres Datenarray mit Zufallszahlen und passen die Balkenhöhe an das neue Datenarray an. Hierzu einfach den Code aus der vorherigen Aufgabe einbinden.

Aufgabe 2 - SVG

  1. Positionieren Sie die eingebundenen Kreise innerhalb der vorgegebenen SVG. Eine zugehörige HTML-Syntax könnte wie folgt lauten: 
    <svg width="500" height="50">
    <circle cx="50" cy="50" r="20"></circle>
</svg>
  2. Passen Sie den Radius der Kreise an die Werte im Datenarray an.

Aufgabe 3 - Balkendiagramm in SVG

  1. Positionieren Sie die Baseline am unteren Rand der SVG.
  2. Übernehmen Sie die Änderung aus der vorhergehenden Aufgabe und fügen Sie einen Farbverlauf entsprechend der Werte des Datenarrays ein.
  3. Übernehmen Sie die Änderung aus der vorhergehenden Aufgabe und fügen Sie dem Balkendiagramm Labels hinzu.
